NumPy

NumPy (Numerical Python) es una de las librer√≠as fundamentales del ecosistema cient√≠fico de Python. Proporciona estructuras de datos y operaciones matem√°ticas altamente optimizadas, esenciales para el procesamiento eficiente de grandes vol√∫menes de datos, algo habitual en inteligencia artificial y machine learning.

A diferencia de las listas nativas de Python, los arrays de NumPy son estructuras homog√©neas y compactas en memoria, lo que permite realizar c√°lculos miles de veces m√°s r√°pidos gracias a la vectorizaci√≥n y al uso de c√≥digo optimizado en C.

Importancia de NumPy en la IA‚Äã

NumPy est√° en la base de casi todas las librer√≠as modernas de IA: Pandas, scikit-learn, TensorFlow o PyTorch se apoyan en sus arrays para representar datos, matrices y tensores.
A continuaci√≥n se muestran dos ejemplos pr√°cticos que ilustran por qu√© NumPy es tan importante en el trabajo con inteligencia artificial.

üß† Ejemplo: Manipulaci√≥n eficiente de datos‚Äã

En inteligencia artificial trabajamos con grandes vol√∫menes de informaci√≥n num√©rica, como im√°genes, se√±ales o series de datos.

Imagina que queremos simular una imagen para entrenar una red neuronal. Cada imagen puede representarse como un array tridimensional, donde:

  • El primer valor indica la altura (n√∫mero de filas de p√≠xeles).
  • El segundo, la anchura (n√∫mero de columnas).
  • El tercero, los canales de color (por ejemplo, RGB = 3 canales).

import numpy as np

# Creamos una imagen aleatoria de 224x224 p√≠xeles con 3 canales (RGB)
# Cada valor representa la intensidad de un color (de 0 a 255)
# Los par√°metros de la funci√≥n randint coinciden con:
# 0 ‚Üí valor m√≠nimo posible.
# 255 ‚Üí valor m√°ximo posible (el 255 no se incluye, por lo que los valores van de 0 a 254).
# (224, 224, 3) ‚Üí forma del array, equivalente a:
# 224 filas (altura de la imagen)
# 224 columnas (anchura)
# 3 canales de color (RGB)
imagen = np.random.randint(0, 255, (224, 224, 3))

# Consultamos su forma: (alto, ancho, canales)
print(imagen.shape)
# (224, 224, 3)

üëâ En este ejemplo, np.random.randint(0, 255, (224, 224, 3)) genera una matriz tridimensional llena de n√∫meros enteros aleatorios entre 0 y 255, imitando los valores de brillo de una imagen real.

El m√©todo .shape nos muestra la estructura del array:

  • 224 filas (alto)
  • 224 columnas (ancho)
  • 3 canales (RGB)

En visi√≥n artificial y redes neuronales, es muy habitual normalizar las im√°genes antes de procesarlas, para que los valores est√©n en un rango m√°s manejable (por ejemplo, entre 0 y 1). Esto ayuda a que el modelo aprenda m√°s r√°pido y de forma m√°s estable.

# Normalizamos todos los p√≠xeles entre 0 y 1
imagen_normalizada = imagen / 255.0

Esta √∫nica l√≠nea divide todos los valores del array a la vez, sin usar bucles for. NumPy aplica la operaci√≥n de forma vectorizada, aprovechando rutinas optimizadas en C que realizan millones de operaciones por segundo.

Este tipo de operaciones masivas y vectorizadas son esenciales en:

  • Visi√≥n por computador (clasificaci√≥n de im√°genes, detecci√≥n de objetos).
  • Reconocimiento facial o de gestos.
  • Entrenamiento de redes neuronales convolucionales (CNN), donde cada imagen de entrada debe normalizarse antes de pasar a la red.

Creaci√≥n de arrays‚Äã

A partir de listas o tuplas (np.array())‚Äã

La forma m√°s b√°sica de crear un array en NumPy es a partir de una lista o tupla de Python.

En el caso del c√≥digo anterior estamos creando un array de una dimensi√≥n:

Tambi√©n pueden crearse arrays bidimensionales (matrices) a partir de una lista de listas:

                   [6, 7, 8, 9, 10]])

Todos los elementos del array tendr√°n el mismo tipo de dato. NumPy lo determina autom√°ticamente, aunque puede especificarse con el par√°metro dtype:

Arrays de ceros, unos y valores aleatorios‚Äã

NumPy ofrece funciones para crear arrays r√°pidamente sin escribir todos los valores manualmente.

np.zeros()‚Äã

Crea un array lleno de ceros del tama√±o indicado.

Par√°metros:

  • shape: tupla que indica el n√∫mero de filas y columnas (por ejemplo, (2, 3) ‚Üí 2 filas y 3 columnas).
  • dtype (opcional): tipo de dato de los elementos (por defecto float).
import numpy as np

# Array unidimensional (vector):
np.zeros(5)
# array([0., 0., 0., 0., 0.])

# Array 2D de 2 filas y 3 columnas
np.zeros((2, 3))
# array([[0., 0., 0.],
#        [0., 0., 0.]])

np.ones()‚Äã

Crea un array lleno de unos. Mismos par√°metros que el m√©todo anterior.

np.ones((3, 2))
# array([[1., 1.],
#        [1., 1.],
#        [1., 1.]])

np.random.rand()‚Äã

Genera n√∫meros aleatorios entre 0 y 1 siguiendo una distribuci√≥n uniforme. A diferencia de np.zeros() o np.ones(), los tama√±os de las dimensiones se pasan como argumentos separados, no dentro de una tupla.

Par√°metros:

  • d1, d2, ...: dimensiones del array (filas, columnas, etc.)
import numpy as np

# Array 2D de 2 filas y 3 columnas
np.random.rand(2, 3)
# array([[0.56, 0.18, 0.72],
#        [0.44, 0.85, 0.02]])

# Array 1D de 5 elementos
np.random.rand(5)
# array([0.12, 0.77, 0.33, 0.90, 0.58])

np.random.randint()‚Äã

Genera n√∫meros enteros aleatorios dentro de un rango especificado. Es muy √∫til para crear datos de ejemplo o simular muestras num√©ricas (por ejemplo, intensidades de p√≠xeles).

Par√°metros:

  • low: valor m√≠nimo (incluido).
  • high: valor m√°ximo (‚ö†Ô∏è excluido ‚Üí el n√∫mero high no aparece).
  • size: forma o dimensiones del array. Puede ser un n√∫mero (para un array 1D) o una tupla (para arrays 2D o superiores).
  • dtype (opcional): tipo de dato (por defecto int).
import numpy as np

# Array 2D de 2 filas y 4 columnas
# con valores aleatorios entre 0 y 9

# array([[3, 7, 2, 9],
#        [5, 1, 0, 8]])

# Array 1D de 5 n√∫meros aleatorios entre 1 y 100

# array([45, 12, 99, 7, 68])

Rangos‚Äã

np.arange()‚Äã

Crea una secuencia de n√∫meros equiespaciados, similar a la funci√≥n range() de Python, pero devolviendo un array de NumPy. Es muy √∫til para generar series num√©ricas o √≠ndices.

Par√°metros:

  • start: valor inicial de la secuencia (incluido).
  • stop: valor final (‚ö†Ô∏è excluido ‚Üí el √∫ltimo valor no se incluye).
  • step: incremento entre valores consecutivos (por defecto 1).
  • dtype (opcional): tipo de dato (entero o flotante).
import numpy as np

# Array de 0 a 8 con paso de 2
np.arange(0, 10, 2)
# array([0, 2, 4, 6, 8])

# Array de 1 a 5 con paso de 1 (por defecto)
np.arange(1, 6)
# array([1, 2, 3, 4, 5])

# Array de n√∫meros decimales
np.arange(0, 1, 0.2)
# array([0. , 0.2, 0.4, 0.6, 0.8])

np.linspace()‚Äã

Crea una secuencia de valores equiespaciados entre un punto inicial y uno final, incluyendo ambos extremos. A diferencia de np.arange(), aqu√≠ no se indica el paso, sino cu√°ntos valores se desean generar.

Par√°metros:

  • start: valor inicial de la secuencia (incluido).
  • stop: valor final de la secuencia (incluido).
  • num: cantidad de valores a generar (por defecto 50).
  • endpoint: si es True (por defecto), incluye el valor final; si es False, lo excluye.
  • dtype (opcional): tipo de dato de los valores generados.
import numpy as np

# 5 valores entre 0 y 1 (incluyendo ambos extremos)
np.linspace(0, 1, 5)
# array([0.  , 0.25, 0.5 , 0.75, 1.  ])

# 4 valores entre 10 y 20
np.linspace(10, 20, 4)
# array([10., 13.33333333, 16.66666667, 20.])

# Excluir el valor final
np.linspace(0, 1, 5, endpoint=False)
# array([0. , 0.2, 0.4, 0.6, 0.8])

üëâ np.linspace() se usa mucho para generar intervalos continuos o ejes de valores en gr√°ficos, funciones matem√°ticas o simulaciones num√©ricas.

Propiedades de los arrays‚Äã

Los arrays de NumPy no solo almacenan datos, tambi√©n guardan informaci√≥n sobre su estructura (forma, dimensiones, tipo de dato, etc.). Estas propiedades son muy √∫tiles en IA y Machine Learning para entender la forma de los datos que vamos a procesar (por ejemplo, el tama√±o de las entradas y salidas de un modelo).

shape‚Äã

Indica la forma del array, es decir, cu√°ntas filas y columnas (y, si aplica, cu√°ntas dimensiones m√°s) tiene. Devuelve una tupla con el n√∫mero de elementos en cada eje.

              [4, 5, 6]])

# (2, 3)

üìò Significado: el array tiene 2 filas y 3 columnas. En arrays tridimensionales (por ejemplo, im√°genes RGB), shape podr√≠a ser (224, 224, 3).

ndim‚Äã

Devuelve el n√∫mero de dimensiones del array (tambi√©n llamado ‚Äún√∫mero de ejes‚Äù).

              [4, 5, 6]])

# 2

üëâ En este caso, a es una matriz 2D. Si fuese un vector, ndim ser√≠a 1; si fuese un tensor de im√°genes, podr√≠a ser 3 o m√°s.

size‚Äã

Indica el n√∫mero total de elementos del array (es decir, el producto de todas sus dimensiones).

              [4, 5, 6]])

# 6

üìò El array tiene 2 filas √ó 3 columnas = 6 elementos.

dtype‚Äã

Muestra el tipo de dato de los elementos del array (int32, float64, bool, etc.). Todos los elementos de un array NumPy son del mismo tipo, lo que mejora el rendimiento.

print(b.dtype)

Estas propiedades son fundamentales en IA para:

  • Verificar la forma de entrada y salida de un modelo.
  • Comprobar si los datos deben reformatearse o redimensionarse.
  • Controlar el tipo de dato (por ejemplo, float32 en TensorFlow para ahorrar memoria).

Acceso y modificaci√≥n de elementos‚Äã

Una de las ventajas de los arrays de NumPy es que permiten acceder, seleccionar y modificar datos de forma r√°pida y flexible, incluso cuando se trata de grandes vol√∫menes de informaci√≥n. Esto resulta esencial en IA, donde necesitamos extraer subconjuntos de datos, dividir muestras o transformar estructuras antes de entrenar modelos.

Acceso por √≠ndice‚Äã

Los elementos de un array se acceden mediante √≠ndices num√©ricos, igual que en las listas de Python, empezando desde 0.

import numpy as np

# 10

# 50

En arrays bidimensionales, se utilizan dos √≠ndices: el primero indica la fila y el segundo la columna.

                   [4, 5, 6]])

print(matriz[0, 1])  # Elemento fila 0, columna 1
# 2

print(matriz[1, 2])  # Elemento fila 1, columna 2
# 6

üëâ Se puede leer as√≠: matriz[fila, columna]

Slicing (rebanado)‚Äã

El slicing permite seleccionar un rango de elementos usando la notaci√≥n [inicio:fin:paso], igual que en las listas, pero tambi√©n puede aplicarse a filas y columnas.

# [20 30 40]

üìò El √≠ndice inicial se incluye y el final se excluye.

En matrices 2D:

                   [4, 5, 6],
                   [7, 8, 9]])

# [[2 3]
#  [5 6]]

üëâ Se est√°n seleccionando las filas 0 y 1 y las columnas 1 y 2. Esta operaci√≥n devuelve un subarray (una porci√≥n del array original).

Modificaci√≥n de elementos‚Äã

Tambi√©n podemos asignar nuevos valores directamente a posiciones o rangos espec√≠ficos.

a[2] = 99
print(a)
# [ 1  2 99  4  5]

En matrices:

                   [4, 5, 6]])

# [[  1 100   3]
#  [  4   5   6]]

Tambi√©n se pueden cambiar varios valores a la vez con slicing:

print(a)
# [ 1 10 20 30  5]

Indexaci√≥n booleana (boolean indexing)‚Äã

La indexaci√≥n booleana permite filtrar elementos de un array seg√∫n una condici√≥n l√≥gica. El resultado es un nuevo array que solo contiene los valores que cumplen la condici√≥n.

# Elementos mayores que 25

print(filtro)
# [False False  True  True  True]

# [30 40 50]

Tambi√©n puede escribirse directamente en una sola l√≠nea:

# [30 40 50]

Podemos usar m√∫ltiples condiciones con operadores l√≥gicos:

  • | (or)

# [20 30 40]

üëâ Esto es muy √∫til en IA para limpiar datos, eliminar valores an√≥malos o seleccionar subconjuntos espec√≠ficos de un dataset.

Subarrays y vistas (views)‚Äã

Cuando seleccionamos una parte de un array (por ejemplo, con slicing), NumPy no crea una copia independiente, sino una vista (view) del array original. Esto significa que si modificamos el subarray, tambi√©n cambia el original.

sub[0] = 999
print(sub)
# [999  30  40]

print(a)
# [ 10 999  30  40  50]

üëâ Cuidado: sub y a comparten los mismos datos en memoria. Si quieres evitarlo, debes crear una copia expl√≠cita:

b[0] = 0
print(b)
# [ 0 30 40]
print(a)
# [ 10 999  30  40  50]

Operaciones b√°sicas y funciones √∫tiles‚Äã

Los arrays de NumPy permiten realizar operaciones matem√°ticas, estad√≠sticas y l√≥gicas de forma vectorizada, es decir, sin necesidad de bucles. Esto los hace ideales para tareas de inteligencia artificial y machine learning, donde se trabajan grandes vol√∫menes de datos y se necesitan c√°lculos r√°pidos y precisos.

Operaciones aritm√©ticas‚Äã

Las operaciones b√°sicas (+, -, *, /, **) se aplican elemento a elemento, tanto entre arrays como con escalares.

import numpy as np

a = np.array([10, 20, 30])
b = np.array([1, 2, 3])

print(a + b)   # Suma
# [11 22 33]

print(a * b)   # Multiplicaci√≥n
# [10 40 90]

print(a / 10)  # Divisi√≥n por escalar
# [1. 2. 3.]
Importante

En las operaciones aritm√©ticas entre arrays (+, -, *, /, etc.), ambos arrays deben tener el mismo tama√±o y forma (shape), o ser compatibles mediante broadcasting.

a = np.array([10, 20, 30])
b = np.array([1, 2, 3])
print(a + b)
# [11 22 33] ‚úÖ Correcto

Si las dimensiones no coinciden y no pueden adaptarse por broadcasting, NumPy mostrar√° un error:

c = np.array([1, 2])
print(a + c)
# ‚ùå ValueError: operands could not be broadcast together

üëâ Antes de realizar operaciones entre arrays, comprueba sus formas con array.shape. Esto es especialmente importante en IA, donde los datos (por ejemplo, vectores de entrada o matrices de pesos) deben tener dimensiones compatibles para poder combinarse correctamente.

Tambi√©n se pueden aplicar funciones matem√°ticas directamente sobre los arrays:

np.sqrt(a)     # Ra√≠z cuadrada
# [3.162 4.472 5.477]

np.exp(b)      # Exponencial
# [ 2.718  7.389 20.086]

np.log(a)      # Logaritmo natural
# [2.302 2.996 3.401]

üëâ Estas operaciones son la base de muchas transformaciones en IA: normalizaci√≥n de datos, c√°lculo de activaciones, errores o funciones de p√©rdida.

Funciones estad√≠sticas‚Äã

NumPy incluye funciones para obtener medidas de resumen sobre los datos. Son muy utilizadas en preprocesamiento, an√°lisis exploratorio y evaluaci√≥n de modelos.

x = np.array([5, 10, 15, 20])

print(np.mean(x))    # Media
# 12.5

print(np.median(x))  # Mediana
# 12.5

# 5.59016994

print(np.sum(x))     # Suma de todos los elementos
# 50

print(np.min(x), np.max(x))   # M√≠nimo y m√°ximo
# 5 20

Estas funciones son esenciales, por ejemplo, para:

  • Calcular medias y desviaciones antes de normalizar los datos.
  • Obtener el rango de valores para aplicar escalado min‚Äìmax.
  • Resumir resultados de predicci√≥n o m√©tricas de rendimiento.
Conceptos matem√°ticos

Media (promedio): Representa el valor medio de un conjunto de datos. Se calcula sumando todos los elementos y dividi√©ndolos entre el n√∫mero total de valores.

media=x1+x2+‚ãØ+xnn\text{media} = \frac{x_1 + x_2 + \dots + x_n}{n}media=nx1‚Äã+x2‚Äã+‚ãØ+xn‚Äã‚Äã

Ejemplo: Para los datos [5, 10, 15, 20], la media es [ (5 + 10 + 15 + 20) / 4 = 12.5 ]

Mediana: Es el valor central cuando los datos est√°n ordenados.

  • Si el n√∫mero de elementos es impar, la mediana es el valor del medio.
  • Si es par, es el promedio de los dos valores centrales.

Ejemplo: Para [5, 10, 15, 20], los dos valores centrales son 10 y 15, por lo que la mediana es ((10 + 15) / 2 = 12.5).

Desviaci√≥n est√°ndar: Mide cu√°nto se alejan los datos de la media. Una desviaci√≥n peque√±a indica que los valores est√°n muy concentrados; una desviaci√≥n grande, que est√°n muy dispersos.

œÉ=1n‚àëi=1n(xi‚àíxÀâ)2\sigma = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(x_i - \bar{x})^2}œÉ=n1‚Äãi=1‚àën‚Äã(xi‚Äã‚àíxÀâ)2‚Äã

Ejemplo: En [5, 10, 15, 20], la media es 12.5. Las diferencias al cuadrado respecto a la media son: ((5-12.5)^2, (10-12.5)^2, (15-12.5)^2, (20-12.5)^2). La ra√≠z cuadrada del promedio de esas diferencias da la desviaci√≥n est√°ndar ‚âà 5.59.

üëâ En IA y Machine Learning, estos conceptos se utilizan para entender la distribuci√≥n de los datos, detectar valores at√≠picos (outliers) y aplicar normalizaci√≥n o estandarizaci√≥n antes del entrenamiento.

Operaciones l√≥gicas y comparaciones‚Äã

NumPy permite realizar comparaciones elemento a elemento, devolviendo arrays de valores booleanos (True/False).

Estas comparaciones pueden combinarse con operadores l√≥gicos:

  • & ‚Üí and

üëâ Las condiciones booleanas se utilizan constantemente para filtrar o validar datos, como ya se vio en el apartado de indexaci√≥n booleana.

Manipulaci√≥n de forma‚Äã

En inteligencia artificial y machine learning, a menudo necesitamos reorganizar los datos para adaptarlos a la forma que espera un modelo. Por ejemplo, una red neuronal puede requerir vectores 1D, matrices 2D o tensores 3D. NumPy facilita este proceso mediante distintas funciones para modificar la forma (shape) de los arrays sin cambiar su contenido.

reshape()‚Äã

Permite cambiar la forma de un array (su n√∫mero de filas y columnas) sin alterar los datos originales. La cantidad total de elementos debe mantenerse constante.

import numpy as np

a = np.array([1, 2, 3, 4, 5, 6])

# Convertir el array 1D en una matriz 2x3
b = a.reshape((2, 3))
print(b)
# [[1 2 3]
#  [4 5 6]]

Tambi√©n puede usarse -1 como valor autom√°tico para que NumPy calcule una de las dimensiones:

c = a.reshape((-1, 2))
print(c)
# [[1 2]
#  [3 4]
#  [5 6]]
Importante

La funci√≥n reshape() no cambia los datos, solo la forma. Sin embargo, las dimensiones deben ser compatibles con el n√∫mero total de elementos. Por ejemplo, un array con 6 elementos no puede convertirse en una forma (4, 2).

flatten()‚Äã

Convierte cualquier array multidimensional en un vector 1D (una copia independiente).

matriz = np.array([[1, 2, 3],
                   [4, 5, 6]])

v = matriz.flatten()
print(v)
# [1 2 3 4 5 6]

üëâ flatten() crea una nueva copia de los datos. Si se modifica el resultado, el original no cambia.

ravel()‚Äã

Hace lo mismo que flatten(), pero devuelve una vista (view) del array original cuando es posible (sin copiar datos).

matriz = np.array([[1, 2, 3],
                   [4, 5, 6]])

v = matriz.ravel()
print(v)
# [1 2 3 4 5 6]

v[0] = 99
print(matriz)
# [[99  2  3]
#  [ 4  5  6]]

üëâ ravel() es m√°s eficiente en memoria, pero hay que tener cuidado: los cambios en el array resultante pueden afectar al original.

transpose() o .T‚Äã

Devuelve la transpuesta del array, es decir, intercambia filas por columnas. Es una operaci√≥n muy com√∫n en √°lgebra lineal, especialmente en redes neuronales y productos punto.

matriz = np.array([[1, 2, 3],
                   [4, 5, 6]])

print(matriz.T)
# [[1 4]
#  [2 5]
#  [3 6]]

Tambi√©n se puede usar el m√©todo:

transpuesta = np.transpose(matriz)
